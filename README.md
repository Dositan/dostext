# DosText: code editor of the future.

### Dostext is:
- Lightweight
- No dependencies
- Simple
- Just type

---

### Running
Clone the repository:

    $ git clone github.com/Dositan/dostext.git

Change directory:

    $ cd dostext

Run:

    $ python -m app
